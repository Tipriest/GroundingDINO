# README.md
我是ubuntu20.04，cuda12.4

## 1. 环境配置
```shell
conda create -n grounding-dino python==3.9
conda activate grounding-dino
conda install pytorch==2.4.0 torchvision==0.19.0 torchaudio==2.4.0 pytorch-cuda=12.4 -c pytorch -c nvidia
pip install pip==22.3.1
pip install "setuptools>=62.3.0,<75.9"

# 设置CUDA_HOME
echo 'export CUDA_HOME=/path/to/cuda' >> ~/.bashrc
source ~/.bashrc

pip install -e .
```

## 2. 下载权重
```shell
mkdir weights
cd weights
wget -q https://github.com/IDEA-Research/GroundingDINO/releases/download/v0.1.0-alpha/groundingdino_swint_ogc.pth
还可以去[这里](https://github.com/IDEA-Research/GroundingDINO/releases)下载另外一个更大的权重到weights文件夹里面
```

## 3. 测试
可以用test.py来测试，看会不会生成test_output.jpg来判断安装是否成功

## 4. 标注yolo
使用`groundingdino_to_yolo.py`来标注yolo，使用`groundingdino_to_yolo.yaml`来控制标注的参数
